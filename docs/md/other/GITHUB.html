  2. 刷新DNS：控制台输入ipconfig/flushdns

Git Key获取

  1. 在git bash中运行ssh-keygen -t rsa -C "git user"
  2. 此时会在C:\Users\jack\.ssh文件夹内生成id_rsa.pub和id_rsa
  3. 复制id_rsa.pub的内容（就是key）
  4. 放置于github -> setting -> SSH keys中（title随意）
